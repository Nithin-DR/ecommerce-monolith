# E-Commerce Application (Monolith → Microservices)

## Overview

This project is a production-style E-Commerce application built using Java, Spring Boot, React.js, and MySQL.

The application will first be developed as a clean monolithic architecture and later migrated into microservices following industry best practices.

## Technology Stack

### Backend
- Java 17
- Spring Boot 3.5.16
- Spring Data JPA
- MySQL
- Maven

### Frontend
- React.js
- Axios
- Bootstrap

### Tools
- IntelliJ IDEA Community
- VS Code
- Docker
- Postman
- Git
